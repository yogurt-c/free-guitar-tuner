# 오디오 엔진 스펙

## 오디오 캡처

- 샘플레이트: 44100Hz
- 버퍼 크기: 2048 샘플
- 채널: 모노
- 패키지: `record`, `audio_session`

## 피치 감지 알고리즘: YIN

단순 FFT 대비 정확도 높음 (±1 cent 오차 수준).
기타 저음현(E2~A2)의 배음 처리에 강함.
Autocorrelation 기반으로 연산 비용 낮음.

## Cents 표시 기준

- 범위: -50 ~ +50 cents
- ±5 이내 → 초록 (합격)
- ±5~15 → 노랑
- ±15 초과 → 빨강

## 안정화 처리

- 노이즈 게이트: 신호 강도 임계값 이하 무시
- 다중 프레임 평균: 연속 N 프레임 주파수 평균으로 떨림 방지
