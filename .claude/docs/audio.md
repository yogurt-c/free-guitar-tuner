# 오디오 엔진 스펙

## 오디오 캡처

- 샘플레이트: 44100Hz
- 버퍼 크기: 2048 샘플
- 채널: 모노
- 패키지: `record`, `audio_session`

## 피치 감지 알고리즘: YIN

단순 FFT 대비 정확도 높음 (±1 cent 오차 수준).
기타 저음현(E2~A2)의 배음 처리에 강함.
Autocorrelation 기반으로 연산 비용 낮음.

## Cents 표시 기준

- 범위: -50 ~ +50 cents
- |cents| < 3 → 초록 inTune (#7DD3A0)
- cents < -3 → 오렌지 flat (#E0824A) — 음정 낮음
- cents > 3 → 파랑 sharp (#5BA3E0) — 음정 높음

## 안정화 처리

- 노이즈 게이트: RMS 신호 강도 0.01 미만 무시
- 다중 프레임 중앙값: 연속 7 프레임 주파수 중앙값으로 떨림 방지 (평균 대비 이상값에 강건)
